# Controle de Gastos em Python

Esse é um projeto simples que fiz para praticar Python enquanto estudo Análise e Desenvolvimento de Sistemas.

A ideia foi criar um pequeno programa no terminal para registrar gastos do dia a dia e depois poder visualizar tudo que foi anotado.

## O que o programa faz

- Permite adicionar um gasto
- Salva o gasto em um arquivo
- Mostra a lista de gastos registrados

## Tecnologias usadas

- Python
- Manipulação de arquivos

## O que aprendi

Fazendo esse projeto consegui praticar:

- uso de funções
- entrada de dados com input
- leitura e escrita de arquivos
- estrutura de repetição

## Melhorias futuras

Algumas ideias que pensei para melhorar:

- separar gastos por categoria
